## Class 2

1. git clone 這台主機要與哪個github下載檔案

2. git status 目前的改動與狀態

3. git add xxxx 將 xxxx 傳上 github

4. git add . 將整個資料夾 (專案) 全部檔案傳上去
      
5. git commit 跳出 vim，並記錄目前最新的動態 (本機)

6. git push 將最新的動態傳到 github (網路)

## 11.23

1.git clone 這台主機要與哪個github上專案連線 
2.git status 目前的改動與狀態
3.git add xxxx 將xxxx傳上github
      add .    將整個資料夾(專案)全部檔案船上去
4.git commit 跳出vim，並記錄目前最新的動態(本機)
5.git push 將最新的動態傳到github(網路)

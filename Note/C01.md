## 11.23

1.git clone 這台主機要與哪個 github 上專案連線

2.git status 目前的改動與狀態

3.git add xxxx 將 xxxx 傳上 github

      add .    將整個資料夾 (專案) 全部檔案傳上去
      
4.git commit 跳出 vim，並記錄目前最新的動態 (本機)

5.git push 將最新的動態傳到 github (網路)
